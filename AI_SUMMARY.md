# AI 生成摘要

**提供商**: OpenAI · gpt-5.4

## 仓库描述

这是一个面向 ChatGPT 官网图片能力的 OpenAI 兼容代理与自托管方案，集成图片生成、编辑、在线工作台和账号池管理，适合想快速搭建绘图接口或研究兼容层实现的人关注。它文档完整、Docker 部署友好，并支持多种存储后端与多架构运行，具备较强的落地性。

## 详细摘要

- 提供兼容 OpenAI 的图片生成、图片编辑、chat/completions 与 responses 接口，便于接入现有客户端或上游系统。
- 内置在线画图工作台与多图编辑能力，支持会话历史、本地保存和服务端缓存，开箱即用。
- 包含较完整的账号池管理能力，如额度刷新、失效剔除、限流检查、批量导入与代理配置。
- 支持 Docker 自托管、多架构镜像以及 JSON、SQLite、PostgreSQL、Git 等存储后端，部署与扩展较灵活。
